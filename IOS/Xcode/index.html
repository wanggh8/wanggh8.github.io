Xcode常用快捷键

代码编辑

代码编辑 Mac
代码左缩进 Command + [
代码右缩进 Command + ]
代码格式化 Control + I
光标移到前一行（previous） Control + P
光标移到后一行（next） Control + N
光标移到行首 Control + A
光标移到行尾（end） Control + E
调换光标两边的字符，并移动光标到两个字符后（transpose） Control + T
删除光标右侧字符 Control + D / Delete
删除本行光标右侧所有字符（kill） Control + K
