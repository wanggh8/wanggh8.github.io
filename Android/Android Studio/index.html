快捷键

Mac OS 外接键盘映射

修饰键 符号 外接键盘
Command Win
Shift Shift
Option Alt
Control Ctrl

Mac OS

  • Command + Shift + F ：在路径中查找
  • Option + Shift + 方向键 ：上下移动单行代码

Windows

常规 Windows/Linux Mac
添加到收藏夹 Alt+Shift+F Option+Shift+F
打开设置对话框 Ctrl+Alt+S Command+,
打开项目结构对话框 Ctrl+Alt+Shift+S Command+;
在标签页和工具窗口之间切换 Ctrl+Tab Ctrl+Tab

工具窗口

工具窗口 Windows 和 Linux Mac
运行 Shift+F10 Ctrl+R
调试 Shift+F9 Ctrl+D
项目窗口 Alt+1 Command+1
Logcat窗口 Alt+6 Command+6
版本控制窗口 Alt+9 Command+9
隐藏所有工具窗口 Ctrl+Shift+F12 Command+Shift+F12
焦点返回编辑器 Esc Esc

样式和格式

样式和格式 Windows 和 Linux Mac
重新格式化代码 Ctrl+Alt+L Option+Command+L
导航和搜索 Windows 和 Linux Mac
全局搜索内容（包括代码和菜单） 两次 Shift 两次 Shift
全局查找类 Ctrl+N Command+O
全局查找文件 Ctrl+Shift+N Command+Shift+O
全局查找方法或变量 Ctrl+Shift+Alt+N Command+Option+O
全局查找字符 Ctrl+Shift+F Command+Shift+F
当前文件查找字符 Ctrl+F Command+F
当前文件查找字符下一项内容 F3 Command+G
当前文件查找字符上一项内容 Shift+F3 Command+Shift+G
当前文件查找引用当前光标位置处的类、方法、字段、参数或语句的所有代码片段 Alt+F7 Option+F7
打开文件结构弹出式菜单 Ctrl+F12 Command+F12
跳到源代码（声明处） F4 或 Ctrl+Enter F4 或 Command+↓
在新窗口中打开当前编辑器标签页 Shift+F4 Shift+F4
关闭活动编辑器标签页 Ctrl+F4 Command+W
从工具窗口返回到编辑器窗口 Esc Esc
转到具体行 Ctrl+G Command+L
