工作经历

  • 联通（广东）产业互联网有限公司 联通医疗云基地

    负责智慧医疗事业部自研产品云护理 Android 端的研发、 代码优化以及架构重构工作。同时参考阿里 Android开发手册和项目实践，制定了Android 开发规范。在新项目护理管理App Android端，负责项目技术选型、底层框架搭建，并优化了项目封装网络请求框架。此外，兼职产互工会技术创新部副部长、产互工会网球兴趣协会副会长。

    • 智慧医疗事业部重点自研产品 云护理 Android 端

      云护理是一款针对医院护士护理优化和信息化的产品，实现了互联网+护理，医嘱全闭环处理，提高护士工作效率。项目采用原始Java进行开发，使用目前主流 MVP 架构模式，实现视图和模型层的分离。封装Retrofit 2.0 、OkHttp 3 、Rxjava 、Logger 网络请求库。在项目中，我进行了项目文件结构的优化、Office文件预览功能实现、分页加载、上拉加载下拉刷新代码优化和修复等项目工作，实现和完善了口服发药、今日出院、护理知识库、今日待办、医嘱执行、护理执行、摆药配药、检验执行、语音识别等功能模块。目前项目已上线多家医院，称为广东联通省级标杆项目。

    • 智慧医疗事业部孵化产品 护理管理 Android 端

      护理管理 Android 端是云护理项目的延伸，针对 Google 推荐的 MVVM + JetPack 开发，搭建 PureMVVM中台架构，实现了视图层和模型层数据的双向绑定，达到数据驱动页面的设计模式。使用适配器模式、观察者模式、单例模式、模板方法模式等设计模式。使用 Retrofit + Rxjava + Okhttp 主流开源框架，封装异步处理、错误处理和日志框架，重构 Android 网络请求，使其适用 于 RESTful API，改善旧项目 POST 请求走天下的局面。基于 Databinding 和 BindingAdapter 封装适用于数据绑定的 Recyclerview 组件库， 使 Android 开发列表类界面实现更加便利，减少重复代码。

    • 智慧医疗事业部孵化产品 护理管理 iOS 端

      iOS 端护理管理系统采用 Swift 进行开发，初步搭建护理管理系统 iOS 端的登录和主页界面。使用 Alamofire 优秀的网络请求框架、Snapkit 自动布局框架、SwiftyJSON 数据处理框架等优秀的开源框 架。在 UI 的技术选型中，为方便后续多人协同编程，采用目前一线大厂的 UIKit 纯代码 Snapkit 自动布局方案和目前 Apple 推荐的 Swift UI 方案的混合解决方案，摒弃 XIB 和 Storyboard 方案。使用 Cocoa MVC 架构模式和 MVVM 架构模式混合模式，在简单页面中使用 Cocoa MVC，在复杂的界面中，使用数据绑定的 MVVM 架构模式，减少代码的复杂度和代码冗余。

个人项目

  • 2020.10-2021.01 PureMVVM 基于 Databinding 的 Android 架构中台

项目结合 Google 推荐的 JetPack 组件库，引入了 DataBinding、ViewBinding、LiveData、 LifeCycle、 ViewModel、 Navigation导航等新技术。封装了基于数据绑定的通用基类，包括：Activity、Fragment、Recyclerview、 Adapter，并基于开源库 Logger，封装通用的日志处理框架。使用 BindingAdapter 数据绑定适配器，可快速实现视图层根据传入的信息进行定制化显示，并实现代码复用。采用 MVVM 架构模式，可进一步实现视图和数据的分离，更好地实现职责分离，减少 Activity、Fragment、Adapter 的代码量。使用 implementation 'com.wanggh8.pure:puremvvm:1.1.3'引入项目，快速搭建基于数据绑定的 MVVM 架构项目，直接进行业务层功能开发。

  • 2019.03-2019.07 iOS 开发 Feed 信息流应用

该项目是实验室与字节跳动合作的选题之一。Feed 信息流应用是在 iOS 平台下的一款根据字节跳动旗下今日头条实现的 APP，主要功能是登录注册、接收浏览新闻、发布新闻、对新闻发布评论、对新闻点赞、查看新闻详细、图片浮层查看等。负责部分iOS页面的实现和后端开发维护工作。项目采用Objective-C开发，使用绝对布局和Masonry自动布局。网络请求采用AFNetWorking，使用SDWebImage加载网络图片。

  • 2019.10-2020.03 安卓应用开发爱心回收

爱心回收是一款个人独立完成的面向中大校园的安卓平台应用，一方面提供了用户登录注册、社团认证、预约赠送等功能；另一方面提供了设置赠送时间地点、查看管理赠送订单，回馈赠送方等功能。此产品可以有效地帮助大学生参与爱心回收奉献爱心，同时缓解社团的压力。

选定项目采用 MVC 框架，主题采用 XUI 开源项目，网络请求通过 Retrofit2 + OkHttp3 + RxJava 实现，通过 RadioGroup + Fragment 实现底部导航栏。本项目使用 PictureSelector 2.0 开源图片选择器，图片加载使用 Glide。列表采用 RecyclerView 实现，并采用 wasabeef 开源动画组件，实现缩放动画效果。最后采用约束布局，减少嵌套布局加快渲染速度。后端采用 LoopBack 框架，数据库采用PostgreSQL。

  • 2018.11-2019.01 安卓应用开发密码守护者

密码守护者是一个 Android 平台的密码管理软件，可以保存各种密码。每个密码都有自己的分类，名称等信息。通过这些信息可以快速检索。密码是通过 SHA256 加密存储，提供修改密码、删除密码、随机生成密码等功能。目前在考虑 iOS 平台的研发与 Android 平台的重构。

项目采用 MVC 框架，主要负责 Model 层的数据模型、本地数据库存储，采用基于 SQLite 的GreenDao 的第三方开源数据库访问 ORM 框架，存储数据既安全又高效，同时支持加密数据库来保护敏感数据。
